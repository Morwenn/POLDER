1.2. formula — Miscellaneous mathematical formula¶

This module contains miscellaneous mathematical formula is various areas of mathematics. Most of them exist because they are needed in the implementations od other POLDER modules.

1.2.1. Basic functions¶

int math::sign(Number x)¶

Signum function: \(\mathrm{sign}(x) = \begin{cases} -1 & \text{if } x < 0, \\ 0 & \text{if } x = 0, \\ 1 & \text{if } x > 0. \end{cases}\).

CommonType math::sum(T first, U second, Rest... rest)

Return the sum of two or more arguments. If the arguments have different types, the result type will be the common type of the different arguments, std::common_type_t<T, U, Rest...>.

CommonType math::mean(Numbers... args)

Return the mean of two or more arguments. If the arguments have different types, the result type will be the common type of the different arguments, std::common_type_t<Numbers...>.

Number math::sqr(Number x)¶

Return the square of x: \(x*x\).

Number math::clamp(Number x, Number min, Number max)¶

Return x, clamped between min and max.

1.2.2. Integer-related functions¶

bool math::is_even(Integer n)¶

Return whether n is an even number.

bool math::is_odd(Integer n)¶

Return whether n is an odd number.

bool math::is_prime(Unsigned n)¶

Return whether n is a prime number.

Unsigned math::prime(Unsigned n)¶

Return the nth prime number.

Unsigned math::fibonacci(Unsigned n)¶

Return the nth Fibonacci number: \(F_n = F_{n-1} + F_{n-2}\).

Unsigned math::gcd(Unsigned a, Unsigned b)¶

Return the greatest common divisor of a and b.

Unsigned math::lcm(Unsigned a, Unsigned b)¶

Return the least common multiple of a and b.

Unsigned math::modpow(Unsigned a, Unsigned b, Unsigned c)¶

Modular exponentiation function. Computes \(a^b \pmod{c}\) and may prevent overflows.

1.2.3. Angle conversions¶

Float math::degrees(Float x)¶

Converts angle x from radians to degrees.

Float math::radians(Float x)¶

Converts angle x from degrees to radians.

1.2.4. Trigonometric functions¶

Float math::sinc(Float x)¶

Unnormalized cardinal since of x: \(\mathrm{sinc}(x) = \frac{\sin(x)}{x}\).

Float math::normalized_sinc(Float x)¶

Normalized cardinal since of x: \(\mathrm{sinc}(x) = \frac{\sin(\pi x)}{\pi x}\).

1.2.5. Miscellaneous functions¶

std::array<std::complex<Float, 2u>> math::quadratic(Float a, Float b, Float c)¶

Return the results x solutions of the quadratic equation \(ax^2 + bx +c\).

\(x = \frac{-b\pm\sqrt{b^2-4ac\ }}{2a}\).
