1.4. norm — Mathematical norms¶

This module defines tag types that represent mathematical norms. These tags can be used to specify which norm should be used when computing functions such as a vector norm, a matrix norm or the distance between two points.

All of tag types live in the namespace math::norm.

type math::norm::manhattan¶

Tag for the Manhattan norm: \(\|x\|_1 = \sum\limits_{i=1}^n |x_i|\).

type math::norm::euclidean¶

Tag for the Euclidean norm: \(\|x\|_2 = \sqrt{\sum\limits_{i=1}^n x_i^2}\).

type math::norm::maximum¶

Tag for the maximum norm: \(\|x\|_\infty = \max \left(|x_1|, \ldots ,|x_n| \right)\).

type math::norm::p¶

Tag for the p norm: \(\|x\|_p = \left(\sum\limits_{i=1}^n |x_i|^p\right)^{1/p}\).
